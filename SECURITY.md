# Security Policy

**Please do not report security vulnerabilities through public GitHub issues, pull requests, or discussions.**

## Reporting a Vulnerability

Email security reports to **sascha.egerer@flowd.de**. For sensitive details, encrypt your report with the [PGP key](#pgp-key) below.

## Supported Versions

Phirewall is pre-1.0. Only the latest minor release receives security fixes.

| Version | Supported |
| ------- | --------- |
| 0.4.x   | Yes       |
| < 0.4   | No        |

## Coordinated Disclosure

- We will acknowledge your report and work with you on a fix.
- Please keep the issue confidential until a fixed release is published.
- Reporters are credited in the CHANGELOG and GitHub Security Advisory unless they prefer to remain anonymous.
